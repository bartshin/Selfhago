# moody
- 카메라 필터 앱



### moody description
![image](https://user-images.githubusercontent.com/26545623/122565503-85423780-d081-11eb-8a0e-812668660ad7.png)

1. 에디팅 화면
- feature
- 에디팅 완료 후 적용된 프리뷰 화면 제공

2. 메인 화면
- feature
- [중앙 카메라 버튼]
- [edit] [filter] 버튼 클릭 시 해당 페이지 이동
- 우측 하단 버튼 > 카메라 전환 

3. 필터 화면
- feature
- [중앙 카메라 버튼]
- 필터들은 좌우 좌우 슬라이딩 하면서 적용 가능
- 우측 하단 버튼 > 카메라 전환

  - bug-fix/수정한이슈
  - etc..
